DocuMind AI is a premium, browser-based data extraction engine that leverages Google's Gemini 2.5 Flash model to convert unstructured PDF documents into structured Excel spreadsheets. Designed for speed and precision, it allows users to define custom extraction rules or use AI to automatically identify key data points.

# ‚ú® Features

Gemini 2.5 Integration: Uses state-of-the-art LLM capabilities to understand document context.

Hybrid Extraction: Support for both Regex (pattern-based) and AI (natural language) extraction rules.

Client-Side Processing: PDF parsing happens entirely in your browser using pdf.js.

Excel Export: Instant conversion of results into .xlsx format via SheetJS.

Auto-Suggest Rules: Let the AI analyze your document and suggest what data should be tracked.

Modern UI: Built with React and Tailwind CSS for a sleek, dark-mode professional experience.
